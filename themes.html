🎨 Themes Demo

This page demonstrates the themes library. The floating 🎨 icon on the bottom-right opens the collapsible theme panel.

  • Click any theme to apply it
  • Click 🎲 Random to pick a random theme
  • Click 🔁 Shuffle to start auto-rotation (default 18s)
  • Press Ctrl + H to open the help popup

Sample content

This is sample content to show how the theme library affects the entire page background, text color and buttons.

Library exposes a global object window.themesSwitcher with methods like setTheme(name), startShuffle(sec), stopShuffle().
